# Admin Dashboard System

A modern, responsive admin dashboard built with HTML, CSS, and JavaScript. This project features a comprehensive dashboard interface with user management, analytics, and reporting capabilities.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Dashboard**: Real-time statistics, charts, and data visualization
- **Navigation**: Collapsible sidebar with intuitive menu system
- **User Management**: User profile dropdown with settings and logout
- **Dark/Light Mode**: Toggle between color schemes based on preference
- **Search Functionality**: Quick search across the dashboard
- **Data Tables**: Sortable and interactive data tables
- **Notifications**: Notification system with badge indicators

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
- Canvas API for chart visualization
- Font Awesome for icons

## Installation

1. Clone or download the project files
2. Open `index.html` in your preferred web browser
3. Alternatively, serve the files using a local web server:
   ```bash
   cd Admin-Dashboard-System
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000` in your browser

## Usage

- Click the hamburger menu icon to toggle the sidebar
- Use the dark mode toggle in the header to switch color schemes
- Click on table headers to sort data
- Click on user profile to access user options
- Use the search bar to quickly find content

## Dashboard Components

- **Statistics Cards**: Display key metrics with visual icons

- **Recent Orders Table**: Sortable table with order information

- **Navigation Menu**: Collapsible sidebar with dashboard sections

## Customization

The dashboard can be easily customized by modifying:

- Color schemes in `styles.css`
- Chart data in `script.js`
- Menu items in `index.html`
- Dashboard widgets in `index.html`
